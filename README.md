1.2 데이터 설명: 

OpenML(ID: 31)에서 가져온 'credit-g' 데이터를 사용했습니다. 총 1,000명의 데이터가 있고, 나이, 집 소유 여부, 대출 금액 등 20가지 정보가 들어있습니다.


1.3 질문: 

"대출 금액이 크면 신용도가 떨어질까?" 혹은 "대출 기간이 길수록 위험할까?" 같은 궁금증을 해결해보고자 합니다.
